# AI Language Translator

## 📌 Project Description
This project is an AI Language Translator developed using Python and Tkinter. It allows users to translate text between multiple languages using Google Translator.

## 🚀 Features
- Translate text into multiple languages
- Auto language detection
- Copy translated text
- Swap source and target languages
- Save translation to a text file
- Translation history
- Simple and user-friendly GUI

## 🛠 Technologies Used
- Python
- Tkinter
- deep-translator
- pyperclip
- gTTS

## ▶️ How to Run

```bash
pip install -r requirements.txt
python translator.py
```
